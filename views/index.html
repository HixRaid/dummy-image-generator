Sizes

Image size is one of the key characteristics of a graphics file. It is defined in pixels and indicates the number of points that make up the image horizontally and vertically.

Image size has two main components: width and height. For example, if an image is 1920 x 1080 pixels, that means it is 1920 pixels horizontally and 1080 pixels vertically.

Colors

Color in an image is one of the important aspects that determine its visual perception. Color can create a mood, highlight objects, convey information, or evoke emotion in the viewer.

In digital images, color is represented by a combination of three primary colors: red (Red), green (Green) and blue (Blue). This is called the RGB color model. Each pixel in an image contains numerical values for each of these three colors, which determine the intensity of the corresponding color in that pixel.

The intensity of each color is determined by a number from 0 to 255, where 0 represents no color (black) and 255 represents the full intensity of the color (saturated).
